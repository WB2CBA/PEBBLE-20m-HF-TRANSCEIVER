# PEBBLE-20m-HF-TRANSCEIVER
PEBBLE HF is a 20m Monoband HF ALL MODE SDR TRANSCEIVER
![Pebble HF](https://github.com/user-attachments/assets/bb3c359d-36ef-4bb2-bbac-7e968c6b8ea2)

PEBBLE HF is an simple budget HF monoband 20m SDR All Mode Transceiver.

PEBBLE HF is based on USDX Transceiver Project with hardware and Firmware changes adapted to PEBBLE HF.

USDX Transceiver Project Github Link:
https://github.com/threeme3/usdx
